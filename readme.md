## 开发环境：xamp
## 框架：Laravel 5.5
## 必须配置：
- Laravel框架需要的基本配置，看官方文档
- php设置为环境变量
- 安装composer

## 下载后配置：
- composer install
- 复制 .env.example 文件为 .env ,并配置数据库基本内容
- 项目根目录在终端处运行以下命令：
    + php artisan key:generate
    + php artisan migrate

## 运行与调试
- 如果没有服务器，可以下载花生壳，花6块钱实现内网穿透，服务器URL填写花生壳给的域名
- 注册微信公众号-订阅号
- 开发
    + 基本配置：设置服务器配置,并启用
    + 开发者工具
        * 在线接口调试工具
        * 公众平台测试账号：接口配置信息修改，扫测试号二维码。
